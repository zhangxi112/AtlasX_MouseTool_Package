# Release License Files

The release package should include this directory as `licenses/`.

Recommended contents before publishing a final package:

- `LGPL-3.0.txt`
- `GPL-3.0.txt`
- PySide6 / Qt for Python license notice
- Qt third-party license notices
- `THIRD_PARTY_NOTICES.md`
