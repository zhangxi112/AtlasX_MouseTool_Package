## 名称关系说明

本软件正式产品名称为 Atlas-X Cursor Studio。软著登记全称为“Atlas-X Cursor Studio 鼠标指针自定义工具软件 V1.0”，其中“鼠标指针自定义工具软件”为功能描述，“V1.0”为登记版本号。

## 主要功能说明

本软件是一款面向 Windows 桌面环境的鼠标指针自定义工具。软件以普通桌面程序和系统托盘常驻方式运行，主界面包括首页、主题、自定义、设置和关于等页面。用户可以在内置主题列表中查看多套鼠标指针方案，选择后在预览区域查看图形效果、主题名称、说明和可用于按程序切换的主题标识，并可直接将所选主题应用到系统光标。内置方案支持主体色、描边色、强调色和发光色的分项调整，调整后会保存到本地配置，并在预览和实际应用时生效。软件提供自定义鼠标指针图片导入能力，用户可选择 PNG、JPG、JPEG、WEBP 图片，程序会把图片复制到本地工作目录并转换为 RGBA 图像；用户可以选择自动抠图、缩放比例和热点坐标，软件生成带棋盘格背景和热点标记的预览图，确认后生成 Windows 可使用的 `.cur` 光标文件并应用。方案管理方面，软件记录当前生效主题、最近生成的自定义光标路径和内置主题配色，支持按前台程序进程名配置主题切换规则，例如将某个程序映射到指定内置主题或系统默认主题。应用到系统时，软件通过 Windows 光标接口把生成或选中的 `.cur` 文件设置到常用系统光标角色；用户需要恢复时，可在首页或设置中执行恢复系统默认光标，程序调用系统参数接口恢复默认方案并更新本地状态。除光标替换外，软件还包含找回鼠标快捷键、晃动尾迹、点击波纹、动态光标、全屏应用自动暂停增强、全局光标大小调整、开机启动控制和深浅色界面切换。配置保存使用用户目录下的 JSON 文件，默认配置位于仓库 `configs/default_config.json`，运行时配置优先写入 `%APPDATA%/atlas_x_cursor_studio/config.json`，并在权限不足时回退到本地 runtime 目录。异常提示通过统一消息框和状态栏实现，导入图片失败、抠图失败、快捷键格式错误、注册表写入失败、主题应用失败等场景均向用户显示可理解提示；未处理异常钩子会记录日志并尝试恢复系统默认光标。软件适用于多显示器办公、课堂演示、远程会议、投影讲解、深色或复杂桌面背景下的鼠标定位，以及需要临时切换鼠标指针样式的日常 Windows 使用场景。

## 技术特点说明

软件采用 Python 桌面端实现，界面层基于 PySide6/Qt 构建，入口程序创建 QApplication、主窗口和系统托盘管理器，关闭窗口时默认最小化到托盘以保持增强功能运行。核心业务以服务模块拆分：配置管理、光标管理、图片处理、覆盖层显示、运行时增强、托盘热键、开机启动和卸载检测相互独立，由 bootstrap 阶段装配为应用上下文。配置文件采用 JSON 结构，默认配置随程序分发，用户配置保存在 `%APPDATA%/atlas_x_cursor_studio/config.json`，保存时使用 UTF-8 编码并保留最近自定义光标、当前主题、配色和各项开关状态。系统接口调用集中在 Windows 相关服务中，系统光标设置使用 `ctypes.windll.user32` 的 `LoadImageW`、`SetSystemCursor` 和 `SystemParametersInfoW`；开机启动使用当前用户 `HKCU\Software\Microsoft\Windows\CurrentVersion\Run` 注册表项；全局快捷键和前台窗口检测也限定在 Windows 桌面环境。图标资源处理使用 Pillow 生成内置主题图像、预览 PNG 和 `.cur` 文件；自定义图片导入后会进行格式校验、RGBA 转换、透明区域裁剪、缩放居中和热点设置，可选通过 ONNX Runtime/U2Net 进行本地抠图。错误处理采用日志、返回值校验、异常捕获和界面提示结合的方式；未处理异常发生时会尝试恢复系统默认光标，避免用户停留在异常光标状态。打包发布使用 PyInstaller 规格文件收集代码、配置和 assets，发布脚本生成 `dist/AtlasXCursorStudio`、便携 zip 包和 Inno Setup 安装器。
